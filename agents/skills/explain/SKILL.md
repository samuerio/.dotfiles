# Explain

Infer the most appropriate mode from the input. Default to **what** if no strong signal.

- **what** — Explain what it is and what problem it solves.
- **13y** — Explain this in a way a 13-year-old can understand.
- **ascii** — Explain this using ASCII art diagrams.
- **history** — Explain the development history, including the major papers and books involved.
- **implement** — Explain how it is implemented.
- **practice** — Explain how this can be implemented in engineering.
- **shape** — Explain the input/output shape.
